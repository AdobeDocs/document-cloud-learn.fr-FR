# Modification de graphiques dans un PDF

Découvrez comment utiliser le mode de modification dans Acrobat pour ajouter, remplacer, recadrer, redimensionner, faire pivoter et organiser des images directement dans un PDF sur ordinateur et sur le web. Le tutoriel explique également comment générer et affiner des images à partir d’invites textuelles avec le générateur d’images intégré optimisé par Adobe Express. Ce tutoriel vidéo utilise la [nouvelle expérience Acrobat](new-experience.md).

[!BADGE Spécifique à l’équipe]

## Comment le mode Modifier aide différentes équipes

* **Légal**

  Les équipes juridiques utilisent le mode Modifier pour mettre à jour les éléments graphiques et visuels dans les contrats, les politiques et les conseils sans recréer les fichiers, ce qui permet de maintenir les documents exacts et à jour lors des révisions ou des modifications réglementaires.

* **Ventes**

  Les équipes commerciales utilisent le mode Modifier pour remplacer des images, ajuster les mises en page et mettre à jour les visuels des propositions et des documents commerciaux afin d’adapter les documents à des clients ou opportunités spécifiques.

* **H**

  Les équipes RH utilisent le mode Modifier pour réviser les images et les mises en page dans les documents de stratégie, les documents d’intégration et les PDF de formation, afin que les mises à jour puissent être effectuées rapidement sans revenir aux fichiers de conception sources.

* **Finance**

  Les équipes financières peuvent utiliser le mode Édition pour mettre rapidement à jour les graphiques et les visuels dans les PDF financiers, en veillant à ce que les rapports restent exacts et clairs sans revenir au fichier source.

* **Achats**

  Les équipes d&#39;approvisionnement peuvent utiliser le mode Modifier pour mettre à jour rapidement les graphiques des fournisseurs, les images des produits et les détails visuels des propositions ou des contrats des fournisseurs, sans avoir besoin des fichiers d&#39;origine.
