# Repérer les sélections requises avec les groupes de cases à cocher

Découvrez comment utiliser les groupes de cases à cocher pour contrôler le nombre d’options qu’un signataire peut sélectionner sur un formulaire. Les groupes de cases à cocher permettent de s’assurer que les signataires remplissent correctement les documents en appliquant des règles simples, telles que l’exigence d’un nombre minimum ou maximum de sélections, avant qu’un accord puisse être envoyé.

Découvrez comment les cases à cocher sont regroupées :

* Empêcher les envois incomplets ou non valides
* Guider les signataires avec des commentaires clairs et intégrés
* Améliorer la précision des informations collectées dans les formulaires
